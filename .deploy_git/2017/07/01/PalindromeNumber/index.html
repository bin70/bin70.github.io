PalindromeNumber

题目：Determine whether an integer is a palindrome. Do this without extra space.

思路：不能转成字符串来处理，因为不允许额外空间。直接将原数取反后，和原数进行比较。

特殊情况：

  • 负数不是回文数字，因此直接判定FALSE。
  • 只要是回文数字，就不会出现溢出。若溢出，则一定不是回文数字。
